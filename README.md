# iML
i-maruyama lab. による
物理実験用動画像処理ソフトウェア(FitPhysExp)の
開発を行っています。

斜面を転がる物体の運動（二次関数）や振り子の運動（三角関数）を
OpenCV（テンプレートマッチング）を利用して画像認識しエクセル(csv)に出力する事で
関数を可視化するソフトウェアです。
